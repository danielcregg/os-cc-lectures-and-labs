# OS & CC Lectures and Labs

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=flat-square)

Lecture notes and lab guides for the Operating Systems and Cloud Computing (OS & CC) module at ATU. Includes hands-on tutorials covering topics such as WordPress site administration on LAMP stacks.

## Overview

This repository serves as the central collection of written materials for the OS & CC module. It contains step-by-step lab guides and reference documents that accompany the weekly lectures. Current content focuses on building and managing WordPress websites, covering everything from dashboard navigation and theme customisation to plugin management and mobile optimisation.

## Features

- Comprehensive WordPress website building guide with Mermaid diagrams
- Block Editor (Gutenberg) walkthrough for creating pages and custom blocks
- Theme selection and Full Site Editing instructions
- Plugin recommendations for security, SEO, performance, and forms
- Mobile optimisation and troubleshooting checklists
- Best practices and additional learning resources

## Prerequisites

- A web browser for viewing the Markdown guides on GitHub
- Access to a LAMP server with WordPress installed (provided by the instructor for lab sessions)

## Getting Started

### Installation

Clone the repository to access materials offline:

```bash
git clone https://github.com/danielcregg/os-cc-lectures-and-labs.git
cd os-cc-lectures-and-labs
```

### Usage

Browse the guides directly on GitHub for rendered Markdown and Mermaid diagrams, or open the `.md` files in any Markdown editor.

| File | Description |
|------|-------------|
| `how-to-build-a-wordpress-website.md` | Full WordPress site building tutorial |

## Tech Stack

- **Format:** Markdown with Mermaid diagrams
- **Platform:** GitHub (rendered via GitHub Flavored Markdown)
